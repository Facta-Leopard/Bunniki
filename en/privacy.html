Privacy Policy

Last updated: April 13, 2026

Rabbiki is a macOS app that reads user-selected vaults and documents, then writes generated wiki output into a separate output library chosen by the user. Rabbiki does not edit the original source files.

1. Data Rabbiki can access

  • User-selected source files and folders, including Markdown, text, and supported PDF files.
  • User-selected output library paths where Rabbiki writes generated wiki pages, indexes, query results, diagnostics, and logs.
  • Provider configuration data such as selected model names or base URLs.
  • Secrets such as API keys, when provided by the user, stored in the macOS Keychain.

2. How source data is used

Rabbiki uses selected source content to build a searchable index, answer wiki queries, and generate or refresh wiki pages inside the selected output library. Source files remain in place and are not modified by Rabbiki.

3. Remote AI providers

If you choose a remote or networked provider such as OpenAI API, Codex, or a local OpenAI-compatible endpoint running elsewhere on your network, Rabbiki may send prompts, selected wiki context, and supporting extracted text needed to answer a query or draft wiki content to that provider.

If you use Apple Foundation Models on-device, generation stays on the current Mac subject to Apple platform availability.

4. Local storage

  • App settings, sessions, page manifests, and run history are stored locally.
  • Generated wiki pages, logs, and diagnostics are stored only inside the selected output library.
  • API keys or similar credentials are stored in the macOS Keychain, not in the output library.

5. Data sharing

Rabbiki does not operate its own cloud service. Data is shared only with the providers you explicitly configure and use.

6. Your choices

  • You choose which folders and files Rabbiki can access.
  • You choose which provider to use, and you can remove provider credentials at any time.
  • You can reset app data from the Settings screen.

7. Contact

For privacy or support questions, contact shteosis@gmail.com.

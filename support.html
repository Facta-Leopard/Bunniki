Bunniki Support

Official support page for Bunniki questions, bug reports, and App Store support requests for iPadOS.

English

Email: shteosis@gmail.com

Helpful Information to Include

  • Device type, operating system version, and Bunniki app version
  • Provider used, if any
  • Failed action: wiki generation, wiki translation, query, source linking, Provider setup, and so on
  • App language and output language selected in Bunniki
  • Expected behavior, actual behavior, and whether it occurred on iPad, simulator, TestFlight, or App Store build
  • Error messages or screenshots with private or sensitive source text removed

Support Scope

Installation, source attachment, output library setup, wiki generation, wiki refresh, multilingual wiki creation, existing wiki translation, wiki-based query, Chatbot Mode, web wiki view, Provider connection issues, iPadOS advertising display questions, and App Store review follow-up.

Support is provided on a best-effort basis. Please avoid sending sensitive personal data or source documents unless they are strictly needed to reproduce a problem.

설치, 소스 연결, 출력 라이브러리 설정, 위키 생성, 위키 갱신, 다국어 위키 생성, 기존 위키 번역, 위키 기반 질의, 챗봇 모드, 웹 위키 보기, Provider 연결 문제, iPadOS 광고 표시 문의, App Store 심사 후속 문의를 지원합니다.

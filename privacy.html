Bunniki Privacy Policy

Last updated: May 5, 2026

English

Bunniki is an iPadOS app that helps users generate, translate, browse, and query a wiki from local documents and folders selected by the user. It can also answer in Chatbot Mode without searching the local wiki. Bunniki does not scan locations the user has not selected and does not modify original source files.

Data the App Processes

  • Text and file metadata from user-selected source files and folders
  • Generated wiki documents, translated wiki versions, run history, indexes, caches, and diagnostics
  • User questions, wiki context, conversation history, generation instructions, and translation instructions
  • Selected Provider settings, API keys, OAuth tokens, and optional Provider account identifiers
  • Ad-related data processed by Google Mobile Ads SDK on iPadOS when ads are shown

Storage and Providers

Bunniki stores data in the output library selected by the user and in the app support folder. Provider credentials are stored in a local encrypted store using Apple CryptoKit AES-GCM. When OpenAI API, Codex OAuth, or a user-configured OpenAI-compatible network endpoint is used, Bunniki may send the user's question, conversation history, wiki context, generation instructions, translation instructions, excerpts from user-selected documents, and generated wiki page content to that Provider. When a wiki is translated, generated wiki page text may be sent to the selected Provider to create the requested language version. Bunniki discloses network Provider use in the app and asks for consent before using network-based Providers for document, wiki, query, or translation processing. Apple Foundation Models Provider uses on-device models when available.

Advertising and Deletion

The iPadOS version may display ads using Google Mobile Ads SDK. Bunniki does not use source documents, generated wiki pages, user questions, Provider credentials, or local workspace content for advertising, and does not share user-selected document contents with the advertising SDK for ad targeting. Users can delete query history, run history, wikis, source links, Provider credentials, and app state in the app. Privacy questions: shteosis@gmail.com.

한국어

최종 업데이트: 2026년 5월 5일

Bunniki는 사용자가 선택한 로컬 문서와 폴더를 읽어 위키를 생성, 번역, 탐색하고 질의할 수 있게 돕는 iPadOS 앱입니다. 챗봇 모드에서는 로컬 위키를 검색하지 않고 답변할 수 있습니다. Bunniki는 사용자가 선택하지 않은 위치를 임의로 탐색하지 않으며 원본 소스 파일을 수정하지 않습니다.

앱이 처리하는 데이터

  • 사용자가 선택한 소스 파일과 폴더의 텍스트 및 파일 메타데이터
  • 생성된 위키 문서, 번역된 위키 버전, 실행 기록, 인덱스, 캐시, 진단 정보
  • 사용자 질의, 위키 문맥, 대화 기록, 생성 지시, 번역 지시
  • 선택한 Provider 설정, API 키, OAuth 토큰, 선택형 Provider 계정 식별자
  • iPadOS에서 광고가 표시될 때 Google Mobile Ads SDK가 처리할 수 있는 광고 관련 데이터

저장 및 Provider 사용

Bunniki는 사용자가 선택한 출력 라이브러리와 앱 지원 폴더에 데이터를 저장합니다. Provider 인증 정보는 Apple CryptoKit 기반 AES-GCM으로 암호화된 로컬 저장소에 보관됩니다. OpenAI API, Codex OAuth, 사용자가 설정한 OpenAI 호환 네트워크 endpoint를 사용할 경우, Bunniki는 질의, 대화 기록, 위키 문맥, 생성 지시, 번역 지시, 사용자가 선택한 문서의 발췌, 생성된 위키 페이지 본문을 해당 Provider로 전송할 수 있습니다. 위키 번역 시에는 요청한 언어 버전을 만들기 위해 생성된 위키 페이지 본문이 선택한 Provider로 전송될 수 있습니다. Bunniki는 네트워크 기반 Provider를 사용하기 전에 앱 안에서 고지하고 동의를 받습니다. Apple Foundation Models Provider는 사용 가능한 경우 온디바이스 모델을 사용합니다.

광고 및 삭제

iPadOS 버전은 Google Mobile Ads SDK로 광고를 표시할 수 있습니다. Bunniki는 소스 문서, 생성된 위키 페이지, 질의 내용, Provider 인증 정보, 로컬 작업공간 콘텐츠를 광고 목적으로 사용하지 않으며, 사용자가 선택한 문서 내용을 광고 타기팅 목적으로 광고 SDK에 제공하지 않습니다. 사용자는 앱 안에서 질의 기록, 실행 기록, 위키, 소스 연결, Provider 인증 정보, 앱 상태를 삭제할 수 있습니다. 개인정보 문의: shteosis@gmail.com.
